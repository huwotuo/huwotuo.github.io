# 🚂 《[绿皮火车的慢情书](https://huwotuo.github.io/)》

林晓把第三十七封信塞进信封时，火车的鸣笛声刚好从山坳里传过来。墨绿色的车厢在蜿蜒的铁轨上慢慢爬行，像一条沉睡的巨蟒，把载着她思念的信笺，送到千里之外的戈壁滩。

她和陈默是在春运的火车上认识的，他帮她把行李箱架到行李架，指尖碰到她的手背，烫得她赶紧缩回手。后来他们挤在两节车厢的连接处，分享一包原味瓜子，他说他要去边疆当兵，她说她要去南方读大学。火车在黑暗中穿行，窗外偶尔闪过零星的灯火，像谁失落的星子。

陈默的回信总是很慢，有时是半个月，有时是一个月，信封上盖着不同军区的邮戳，信纸里偶尔会夹着一片干枯的骆驼刺，或是一小撮细沙。林晓把这些小东西都放在一个铁盒子里，摆在宿舍的窗台上，像守着一份慢慢长大的秘密。

去年冬天陈默探亲回来，林晓在火车站等他。风很大，她裹紧围巾，看见一个穿着迷彩服的身影从出口跑出来，手里举着一个用易拉罐做的小花篮，里面插着几枝干枯的戈壁花。他的脸被风吹得通红，笑着说“路上冻坏了”，她扑进他怀里，闻到他身上淡淡的风沙味和肥皂香。

现在林晓还在写信，只是信封上的地址变成了市里的部队家属院。陈默每天傍晚都会骑着自行车去邮局接她，车把上挂着她最爱吃的草莓糖葫芦。夕阳把他们的影子拉得很长，车轮碾过铺满落叶的柏油路，发出沙沙的声响，像一首没有旋律的歌。

金句： 真正的爱情，从来不怕时间和距离的考验，它会在等待中慢慢沉淀，酿成最醇厚的酒。
